## Особенности

- **Русский синтаксис**: Все ключевые слова и операторы на русском языке
- **Си-подобный синтаксис**: Знакомая структура для программистов, привыкших к C/C++
- **Типизация**: Поддержка основных типов данных (число, текст, логический, списки, словари)
- **Переменные**: Объявление и использование переменных
- **Условия**: Конструкции `если...иначе`
- **Циклы**: Циклы `пока` и `для`
- **Функции**: Определение и вызов функций с параметрами и возвращаемыми значениями
- **Структуры данных**: Поддержка списков (массивов) и словарей (ассоциативных массивов)
- **Индексация**: Доступ к элементам списков по индексу и словарей по ключу
- **ООП**: Классы с приватными и публичными полями и методами, конструкторы, наследование
- **Инкапсуляция**: Контроль доступа через модификаторы "приватный" и "публичный"
- **Цепочки вызовов**: Поддержка длинных цепочек вызовов методов
- **Комментарии**: Однострочные комментарии с `//`

## Синтаксис

### Типы данных

- `число` - целые числа (i64)
- `текст` - строки
- `логический` - булевы значения (`истина`, `ложь`)

### Переменные

```goida
// Объявление переменной без типа
имя = "Иван";

// Объявление с указанием типа
возраст:число = 25;
активен:логический = истина;

// Присваивание
возраст = 30;
```

### Арифметические операции

```goida
a = 10;
b = 5;

печать(a + b);  // 15
печать(a - b);  // 5
печать(a * b);  // 50
печать(a / b);  // 2
печать(a % b);  // 0
```

### Логические операции

```goida
результат = (a > b) && (b > 0);
печать(a == b || a != b);
печать(!результат);
```

### Условия

```goida
если (возраст >= 18) {
    печать("Совершеннолетний");
} иначе {
    печать("Несовершеннолетний");
}
```

### Циклы

#### Цикл `пока`

```goida
счетчик = 0;
пока (счетчик < 5) {
    печать("Счетчик: " + счетчик);
    счетчик = счетчик + 1;
}
```

#### Цикл `для`

```goida
для (i = 1; 10) {
    печать("Число: " + i);
}
```

### Функции

```goida
// Функция без возвращаемого значения
функция приветствие(имя:текст) {
    печать("Привет, " + имя + "!");
}

// Функция с возвращаемым значением
функция сумма(a:число, b:число):число {
    вернуть a + b;
}

// Рекурсивная функция
функция факториал(n:число):число {
    если (n <= 1) {
        вернуть 1;
    } иначе {
        вернуть n * факториал(n - 1);
    }
}

// Использование функций
приветствие("Мир");
результат = сумма(5, 3);
факт = факториал(5);
```

### Ввод данных

```goida
пользовательский_ввод = ввод();
печать("Вы ввели: " + пользовательский_ввод);
```

### Комментарии

```goida
// Это однострочный комментарий
переменная = 42; // Комментарий в конце строки
```

## Примеры программ

### Hello World

```goida
печать("Привет, мир!");
```

### Калькулятор

```goida
a = 10;
b = 5;

печать("a + b = " + (a + b));
печать("a - b = " + (a - b));
печать("a * b = " + (a * b));
печать("a / b = " + (a / b));
```

### Проверка четности

```goida
функция четное(число:число):логический {
    вернуть число % 2 == 0;
}

n = 7;
если (четное(n)) {
    печать(n + " четное");
} иначе {
    печать(n + " нечетное");
}
```

### Таблица умножения

```goida
функция таблица_умножения(число:число) {
    для (i = 1; 10) {
        печать(число + " * " + i + " = " + (число * i));
    }
}

таблица_умножения(7);
```

### ООП (Объектно-ориентированное программирование)

#### Определение класса

```goida
класс Человек {
    // Приватные поля (по умолчанию)
    приватный имя: текст = "Неизвестный";
    приватный возраст: число = 0;
    
    // Публичные поля
    публичный активен: логический = истина;
    
    // Конструктор
    публичный конструктор создать(имя: текст, возраст: число) {
        это.имя = имя;
        это.возраст = возраст;
    }
    
    // Публичные методы
    публичный функция получить_имя(): текст {
        вернуть это.имя;
    }
    
    публичный функция получить_возраст(): число {
        вернуть это.возраст;
    }
    
    // Приватные методы (недоступны извне)
    приватный функция секретная_операция(): текст {
        вернуть "Это секретно!";
    }
}
```

#### Создание объектов и работа с ними

```goida
// Создание объекта
человек = новый Человек("Иван", 25);

// Доступ к публичным полям
человек.активен = ложь;
печать(человек.активен);

// Вызов методов
печать(человек.получить_имя());
печать(человек.получить_возраст());
```

#### Пример с цепочками вызовов

```goida
класс Калькулятор {
    публичный результат: число = 0;
    
    публичный функция сложить(а: число, б: число): число {
        это.результат = а + б;
        вернуть это.результат;
    }
    
    публичный функция получить_результат(): число {
        вернуть это.результат;
    }
}

// Использование
калькулятор = новый Калькулятор();
калькулятор.сложить(5, 3);
печать(калькулятор.получить_результат()); // 8
```

## Ключевые слова

| Русское       | Английский аналог | Описание                      |
|---------------|-------------------|-------------------------------|
| `пусть`       | `let`/`var`       | Объявление переменной         |
| `если`        | `if`              | Условие                       |
| `иначе`       | `else`            | Альтернативная ветка условия  |
| `пока`        | `while`           | Цикл с предусловием           |
| `для`         | `for`             | Цикл со счетчиком             |
| `функция`     | `function`        | Объявление функции            |
| `вернуть`     | `return`          | Возврат значения из функции   |
| `печать`      | `print`           | Вывод на экран                |
| `ввод`        | `input`           | Ввод данных                   |
| `истина`      | `true`            | Логическое истина             |
| `ложь`        | `false`           | Логическое ложь               |
| `число`       | `int`/`number`    | Числовой тип                  |
| `текст`       | `string`          | Строковый тип                 |
| `логический`  | `bool`            | Логический тип                |
| `класс`       | `class`           | Объявление класса             |
| `новый`       | `new`             | Создание объекта              |
| `приватный`   | `private`         | Приватный модификатор доступа |
| `публичный`   | `public`          | Публичный модификатор доступа |
| `конструктор` | `constructor`     | Объявление конструктора       |
| `это`         | `this`            | Ссылка на текущий объект      |

## Ограничения

Текущая версия имеет следующие ограничения:

- Нет обработки ошибок времени выполнения
- Нет наследования классов (планируется в будущих версиях)
- Нет статических методов и полей в классах
